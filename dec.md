# RTU102 Packet Decode Example

Разбор конкретного входящего пакета от RTU102 для IMEI `867724030459827`.

Ключ шифрования для этого примера:

```text
1234567891234567
```

Важно: в конфигураторе устройства это `16` ASCII-символов. На сервере в этом репозитории такой ключ тоже можно указывать как `16` ASCII-символов.

## Входной пакет

Размер UDP-датаграммы: `496` байт.

Структура на транспортном уровне:

- старт кадра: `0xC0`
- конец кадра: `0xC2`
- тело между ними после unstuffing: `488` байт
- IMEI: первые `8` байт тела, `uint64 little-endian`
- ciphertext: оставшиеся `480` байт
